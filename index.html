HTML

  • The head element contains information about the webpage.
  • Elements in the head include:
    • title - title of the webpage
    • style - defines CSS style
    • base - defines base URL
    • meta - defines various metadata including character set, descrition, keywords, author, and viewport
    • script - embeds client-side scripts in HTML doc
  • The body element represents the visible content shown to the user.
  • Elements in the body include:
    • header is at the top of the page, usually with navigation menu and/or intro content
    • h1 through h6 - header
    • section - helps create sections
    • p - paragraph or block of text
    • ul (bulleted list), ol (numbered list), and li (list items) - these can even be nested! ;P
    • img - image, with scr = location of image, and alt = text string to describe appearance and functionality of image
    • br - creates a line of empty space
    • hyperlinks using "a" - creates the link to a webpage with href pointing to the URL
    • footer - at the bottom of the page

Git

  • Enter your Git notes here.
  • Basic Git Commands:
    • git status = shows which branch I am in
    • Add a branch and switch to it: git checkout -b BRANCH-NAME
    • checkout moves you to the new working branch
    • -b flags the new branch
    • git add -A = add mod to current working branch, -A adds all current changes
    • git commit -m "name of what to commit"
    • git pull origin main = pulls down any changes to main branch (ALWAYS do this before starting work.)
    • git push origin BRANCH-NAME - pushes your branch
    • git checkout main = goes back to main branch
